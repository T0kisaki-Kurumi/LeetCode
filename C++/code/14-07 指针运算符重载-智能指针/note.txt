1. 在栈上创建对象：Person p1;   创建对象指针时需要另外写：Person *p = &p1;
   在堆上创建对象：Person *p = new Person();
   栈内存是程序自动管理的，不能使用 delete 删除在栈上创建的对象；堆内存由程序员管理，对象使用完毕后要通过 delete 删除。
   在实际开发中，new 和 delete 往往成对出现，以保证及时删除不再使用的对象，防止无用内存堆积。
2. 利用栈内存由程序自动管理的特性，可以借助一个创建在栈上的智能指针对象来托管在堆上创建的对象的释放。
3. ->运算符重载后原本应该写sp->->func(); 但编译器简化为sp->func(); 和直接使用原来的指针调用保持一致。
4. 注意->和*重载时的不同，->重载时返回指针，*重载时返回对象的引用。
