1. 占位参数是函数的一个形参，只写类型，不写变量名，因此在函数内无法使用，但调用函数时必须传入一个实参。
2. 占位参数可以和默认参数一起使用，比如：void func(int x, int = 2)。
3. 占位参数用法：
    ①（ChatGPT）保持接口一致性、用于未来扩展、代码调试
    ②（常见用途）用于重载自增（减）运算符时区分前置和后置，带占位参数的为后置，不带的为前置。
4. 一般来说，前置自增运算符返回对象自身的引用，后置自增运算符返回一个拷贝的临时对象（因为要先返回值再自增）。
5. 不用考虑后置++的链式编程问题，因为原本的后置++运算符就不能链式编程。
6. 函数接受临时对象时如果以引用的形式接受，前面一定要加const。
