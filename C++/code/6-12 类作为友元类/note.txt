1.在类内声明另一个类作为友元类，该友元类就可以访问当前类内的私有内容。
2.友元关系不能被继承
3.友元关系是单向的，A类声明B类为友元类，除非B也声明A为B的友元类，否则A中还是无法访问B中的私有内容。
4.友元关系不具有传递性，B类是A类的友元类，C类是B类的友元类，但是C类不能访问A类中的私有内容。







友元函数声明方法：直接在类内使用friend关键字，再加上全局函数的声明：
    friend class friendClassName;



B * b = new B; 和 B * b = new B(); 的区别：
两者在效果上基本没有区别，只不过前者默认使用无参构造函数，而后者显式地使用了无参构造函数。
两者的"="含义是不同的。new B是一个指向对象的指针，而new B()是一个对象，因此前者的"="可以理解为赋值，后者的"="理解为让一个指针指向一个对象。
