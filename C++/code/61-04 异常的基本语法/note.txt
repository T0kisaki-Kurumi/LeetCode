1. 异常必须有人处理，否则程序会调用一个terminate函数，中断程序
2. 如果没有异常，不会走catch语句
3. 如果不想处理，可以写个throw继续往外抛出当前异常，让上层处理





异常和捕获异常语法
    try{
        // 需要处理异常的代码块
        throw ***;
    }
    catch(int){
        处理int类型异常
    }
    catch(double){
        处理double类型异常
    }
    // 其他类型异常
    catch(...){
        // 如果不想处理，可以写个throw继续往外抛出当前异常，让上层处理
        throw;

        处理其他类型异常
    }
