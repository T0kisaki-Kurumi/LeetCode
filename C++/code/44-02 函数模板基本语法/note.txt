1. 函数模板的两种使用：
    ① 自动类型推导：必须让编译器推导出一致的T，才能使用模板
    ② 显式指定类型：显式地指定T的类型，可以进行隐式类型转换（例如char转int），但是如果不能转（例如char转int&），还是会报错
2. 模板必须要知道T的类型才可以调用，或是通过自动类型推导，或是通过显式指定，否则不能使用。
   一种常见的情况是函数模板没有形参，此时只能显式指定类型，否则编译器不知道T是什么类型。
3. 推导出的T必须要一致，否则报错（例如第一个T是int，第二个T是char）。


函数模板语法：
template<typename T>
void func(T a){

}

自动类型推导： func(a);
显式指定类型： func<int>(a);

只有template后面紧跟的函数是函数模板。万能数据类型可以起随便什么名字，但是一般用T。
